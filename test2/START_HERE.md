## 🎯 ทดลองฟีเจอร์

### 1️⃣ เพิ่มงานใน Timeline
```
1. คลิกเมนู "Timeline"
2. คลิก "+ เพิ่มงาน"
3. กรอกชื่องาน: "ทดสอบระบบ"
4. เลือกวันที่เริ่ม-จบ
5. คลิก "บันทึก"
```

### 2️⃣ สร้าง To-Do List
```
1. คลิกเมนู "To-Do List"
2. พิมพ์: "ซื้อของ"
3. กด Enter
```

### 3️⃣ จดบันทึก
```
1. คลิกเมนู "Notes"
2. คลิก "+ โน้ตใหม่"
3. กรอกหัวข้อและเนื้อหา
4. คลิก "บันทึก"
```

### 4️⃣ ค้นหาข้อมูล
```
1. พิมพ์ในช่องค้นหาด้านบน
2. คลิกผลลัพธ์ที่ต้องการ
```

## 💾 การจัดเก็บข้อมูล

### ข้อมูลเก็บที่ไหน?
ข้อมูลทั้งหมดเก็บใน **LocalStorage** ของ Browser:
- ✅ ข้อมูลจะอยู่ตลอด แม้ปิด Browser
- ⚠️ ข้อมูลจะหายถ้า: ล้าง Cache หรือถอนการติดตั้ง Browser

### วิธี Backup ข้อมูล
```
1. คลิกเมนู "Settings"
2. คลิกปุ่ม "Backup ข้อมูล"
3. ไฟล์ .json จะถูกดาวน์โหลด
4. เก็บไฟล์ไว้ปลอดภัย
```

### วิธี Restore ข้อมูล
```
1. คลิกเมนู "Settings"
2. คลิกปุ่ม "Import ข้อมูล"
3. เลือกไฟล์ .json ที่เคย Backup ไว้
4. ข้อมูลจะกลับมา
```

## ❓ คำถามที่พบบ่อย

### Q: ทำไมโปรแกรมไม่ทำงาน?
**A**: ตรวจสอบว่า:
1. ไฟล์ทั้งหมดอยู่ในโฟลเดอร์เดียวกันหรือไม่
2. มีไฟล์ `core.js` หรือไม่
3. เปิดใน Browser ที่รองรับ (Chrome, Edge, Firefox)

### Q: ข้อมูลหายไปไหน?
**A**: 
- ข้อมูลอาจหายถ้าล้าง Cache ของ Browser
- แนะนำให้ Backup ข้อมูลเป็นประจำ

### Q: ต้องการแก้ไขโปรแกรมยังไง?
**A**: 
1. อ่าน **GUIDE_ADD_PAGE.md** สำหรับเพิ่มหน้าใหม่
2. อ่าน **GUIDE_CRUD.md** สำหรับจัดการข้อมูล
3. ใช้ Text Editor เช่น VS Code

### Q: มี Tutorial วิดีโอหรือไม่?
**A**: 
- ยังไม่มี แต่เอกสารครบทุกอย่าง
- ลองอ่าน **QUICK_START.md** จะเข้าใจทันที
